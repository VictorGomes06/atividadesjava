align-items:center => Utilizado para centralizar na vertical

Exercício 02

justify-content: center => para centralizar os itens. align-items: center => para retirar as bordas de cima e de baixo.

Exercício 03

justify-content: end => mover os itens para o lado direito. align-items: flex-end => mover os itens para baixo.

Exercício 04

justify-content: space-between => primeiro ítem junto à borda inicial da linha, o último junto à borda final da linha. align-items: center => para retirar as bordas de cima e de baixo.

Exercício 05

align-items: center => para retirar as bordas de cima e de baixo. justify-content: space-around => os ítens são distribuídos na linha com o mesmo espaçamento entre eles.

Exercício 06

align-items: center => para retirar as bordas de cima e de baixo. justify-content: space-evenly => os ítens são distribuídos de forma que o espaçamento entre quaisquer dois itens da linha (incluindo entre os ítens e as bordas) seja igual.
